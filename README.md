# hassos-apps Repository

Home Assistant addon repository by hassos-apps.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the menu (⋮) → **Repositories**
3. Add: `https://github.com/hassos-apps/repository`
