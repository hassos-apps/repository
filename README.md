# hassos-apps Repository

Home Assistant app repository by hassos-apps.

## Installation

1. In Home Assistant, go to **Settings → Apps → App Store**
2. Click the menu (⋮) → **Repositories**
3. Add: `https://github.com/hassos-apps/repository`
